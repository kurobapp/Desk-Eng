# 英単語帳 学習アプリ (ID: 776〜875)

Webブラウザで動作する、シンプルな英単語学習アプリケーションです。
スマホやPCで手軽に「単語一覧での暗記」や「テスト形式での演習」ができます。

## 📁 ディレクトリ構成

```text
.
├── index.html          # アプリの入口（これをブラウザで開く）
├── style.css           # デザイン設定
├── script.js           # アプリの動作ロジック（記述式判定などはここ）
└── data.js             # 【重要】単語データはすべてここにあります
```

## 🚀 使い方

1. `index.html` をブラウザ（Chrome, Safari, Edgeなど）で開いてください。
2. 画面上部のボタンで学習スタイルを自由に切り替えられます。

### 🔄 出題方向の切り替え
* **英 → 日**: 英単語を見て、日本語の意味を答えるモードです（4択形式）。
* **日 → 英**: 日本語を見て、英単語のスペルを入力するモードです（**記述式**）。

### 📖 学習モード
* **問題一覧 (暗記)**: 単語リストを表示します。タップすると答えや例文が開きます。
* **一問一答 (テスト)**: ランダムに出題されるテストモードです。間違えた問題だけを復習する機能もあります。

## 🛠 単語の追加・修正について (Pull Request)

単語の追加や、意味・例文の修正を行いたい場合は、**Pull Request (PR)** を送っていただけると助かります！

### 修正の手順

データはすべて **`data.js`** というファイルに集約されています。

1. このリポジトリを **Fork** してください。
2. **`data.js`** を開き、`wordData` 配列の中身を修正・追加してください。
   ```javascript
   // 例: data.js
   { 
     id: "S1-1", 
     cat: "S1", 
     q: "prove",               // 英単語
     ans_j: "証明する",         // 日本語訳
     example: "The experiment proved..." // 例文
   },
   ```
3. 修正内容をコミットし、**Pull Request** を作成してください。
4. 内容を確認し、マージさせていただきます。

※ カテゴリ（Session）を増やす場合は、`data.js` 上部の `categoryList` にも追記してください。
